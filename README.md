# Portal Universitário - C#

Sistema console em C# que simula um portal acadêmico completo.

## Funcionalidades
- Login do aluno
- Menu interativo
- Matrícula em disciplinas
- Visualização de horário
- Boletim completo
- Atualização de dados pessoais
- Lista de professores
- Declaração de matrícula

## Tecnologias
- C# 
- Programação Orientada a Objetos (POO)
- Classes separadas
- List<T>
- LINQ (em breve)

## Como rodar
```bash
dotnet run
